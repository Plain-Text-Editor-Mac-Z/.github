## 🔍 Overview

<a href="#plainTextEditor">Plain Text Editor</a> is a minimal text editor that works exclusively with plain text — removing all formatting, rich text complexity, and interface clutter that general-purpose text editors include, serving users who simply want to write text or strip formatting from pasted content without any of those features interfering.

The <a href="#plainTextEditor">automatic plain text paste</a> strips all formatting from any content pasted into the editor — when text is copied from a web page carrying font, color, bold, link, and size formatting and pasted into Plain Text Editor, all of that formatting disappears and only the raw text content arrives. This stripping behavior solves the specific frustration of copying text from formatted sources and having that formatting carry through to other applications when the text is subsequently pasted. Using Plain Text Editor as an intermediate clipboard scrubber means text always arrives at its destination as clean plain text with no hidden formatting from the source. The <a href="#plainTextEditor">distraction-free interface</a> provides only the text editing area without toolbars, format menus, sidebar panels, ruler bars, or any other interface elements that are irrelevant when the only goal is typing or reading plain text.

The <a href="#plainTextEditor">font customization</a> controls the typeface, size, and line height of the text display — these are rendering preferences that affect only the visual appearance in the editor without embedding any formatting in the text content itself. The display can be comfortable to read and work in without the text carrying any formatting when copied or saved. The <a href="#plainTextEditor">always-on-top window option</a> keeps the Plain Text Editor window floating above other windows — maintaining a persistent reference note, a list of items to work through, or any text content that should remain visible while working in other applications, without requiring window management attention to keep it in view.

<a href="#plainTextEditor">System sharing integration</a> uses the standard macOS share sheet to send text content to other applications, services, and sharing destinations — sharing to Notes, Mail, Messages, or any share extension without an intermediary export step. <a href="#plainTextEditor">Word and character count</a> displays in the status area for writers tracking output length or working against word count requirements — available at a glance without any additional action.

<a href="#plainTextEditor">Dark mode support</a> follows the macOS system appearance setting automatically — a comfortable dark writing environment when the system is in dark mode without manual theme switching. <a href="#plainTextEditor">Free from the Mac App Store</a> by Sindre Sorhus, a prolific Mac developer known for quality focused utilities — free, actively maintained, and compatible with each macOS update as Apple evolves the platform. The <a href="#plainTextEditor">iCloud document sync option</a> saves the current text to an iCloud-synced document for access from other Apple devices when content needs to persist beyond the current session.

## ❓ FAQ

<details>
<summary>Does Plain Text Editor strip formatting from pasted text?</summary>

Yes. Text pasted into Plain Text Editor has all formatting removed — font, size, color, bold, italic, links, and all other rich text attributes are stripped, leaving only the plain text content.

</details>

<details>
<summary>Can I save text to a file?</summary>

Yes. Text saves to a plain text file through the standard Save dialog, and iCloud document sync enables access from other Apple devices.

</details>

<details>
<summary>Is Plain Text Editor free?</summary>

Yes. Plain Text Editor is free from the Mac App Store with no in-app purchases or subscriptions.

</details>

<details>
<summary>Does it support dark mode?</summary>

Yes. Plain Text Editor follows the macOS system appearance setting automatically without requiring manual switching.

</details>

<details>
<summary>What is the always-on-top window option?</summary>

The always-on-top option keeps the Plain Text Editor window floating above all other application windows, making its content persistently visible while working in other applications.

</details>

<details>
<summary>Does Plain Text Editor count words?</summary>

Yes. Word and character count display in the status area at the bottom of the window.

</details>

<details>
<summary>Who develops Plain Text Editor?</summary>

Sindre Sorhus, a prolific Mac and iOS developer known for quality focused utilities.

</details>

<details>
<summary>Can I share text directly from Plain Text Editor?</summary>

Yes. The standard macOS share sheet sends text to Notes, Mail, Messages, and any available share extension.

</details>

<details>
<summary>Does Plain Text Editor require internet access?</summary>

No. Plain Text Editor works completely offline for all basic text editing functions. iCloud sync requires connectivity when saving to iCloud.

</details>
